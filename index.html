Experience

Portfolio

The portfolio that I display is several projects that I have created individually or as a team.

PenMat

The PenMat website is a program that I created individually with the aim of accommodating complaints from people around the RT/RW. using Boostrap 5, HTML, PHP, Javascript and Mysqli database.

Launta

The Launta website is a program that I created which individually aims to make it easier for a laundry business to record transactions, employees, branches, and laundry equipment. using Boostrap 5, HTML, PHP, Javascript and Mysqli database.

PeduliSehat

The PeduliSehat website is a program that I created which aims to record a person's journey and check whether a person is sick through input of body temperature, using Bootstrap 5, HTML, JavaScript, PHP, and MySQL database.

Go-Camp

Go-Camp is a desktop-based application program that I worked on as a team aimed at providing a means of borrowing and renting camping equipment in the Jember area, using C# and the PostgreSQL database.
