Project Terbaru

Portofolio yang saya tampilkan merupakan beberapa project yang pernah saya buat secara individu saat masih belum kuliah.

Pengaduan Masyarakat

Website Pengaduan Masyarakat merupakan sebuah program pernah saya buat yang bertujuan untuk menampung aduan Masyarakat sekitar rt/rw dengan menggunakan boostrap 5, bahasa pemrograman PHP, dan database Mysqli.

LaundryKita

Website LaundryKita merupakan sebuah program pernah saya buat yang bertujuan untuk membuat data, outlet, transaksi pelanggan disebuah perusahaan Laundry dengan menggunakan boostrap 5, bahasa pemrograman PHP, dan database Mysqli.

PeduliSehat

Website PeduliSehat merupakan sebuah program pernah saya buat yang bertujuan untuk mencatat perjalanan seseorang dan mengecek apakah seseorang sedang sakit melalui suhu badan dimasukan, dengan menggunakan boostrap 5, bahasa pemrograman PHP, dan database Mysqli.

Voting

Website Voting merupakan sebuah program pernah saya buat yang bertujuan untuk melakukan sebauh voting, khsusnya saat pemilihan ketua osis di Smkn 3 Jember dengan menggunakan boostrap 5, bahasa pemrograman PHP, dan database Mysqli.
